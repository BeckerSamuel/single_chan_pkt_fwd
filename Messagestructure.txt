01 23 45 67 89 AB 33 F3 -> Device Id (8 Byte) (use Wifi Mac)
          	     11 -> Device Type (1 Byte)
          	     12 -> Message (1 - 244 Bytes)
       		  12 34 -> Checksum (2 Bytes) (supports up to 257 characters, when every character is 255)

Maximum message size is 255 Byte.

Every byte will be encrypted with the crypting key (the key will added to the byte and for decryption it will be subtracted, all operation are % 255).
